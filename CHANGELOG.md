## 프로젝트 개요

**Piona's AI HUB**는 AI 및 디자인 도구를 카테고리별로 큐레이션한 단일 HTML 대시보드입니다.  
외부 의존성 없이 순수 HTML · CSS · JavaScript(+localStorage)로 구성됩니다.

| 총 서비스 수 | **143개** |

| 배포 | GitHub (`main` 브랜치) |

## 주요 기능

### 네비게이션
- **상단 탭**: 카테고리 전환, 각 카테고리 accent 색 동기화
- **좌측 사이드바**: 소분류 필터 (전체 보기 / 개별 소분류)
- **검색**: 서비스명 · 설명 · 도메인 실시간 검색

### 카드 레이아웃
- **Bento Grid** (4열 기본, 반응형 축소)
- 7번째 카드마다 Wide 카드 (2열) 자동 적용
- 첫 번째 카드 Featured 그라디언트 강조

### 방문 카운터
- 위치: 좌측 사이드바 하단 고정
- 방식: `localStorage` 기반 누적 카운트
- 표시: 카테고리 변경 시에도 항상 유지

### 카테고리 Ambient
- 카테고리 전환 시 배경 방사형 그라디언트 색상 변경
- CSS 변수(`--a`, `--a2`, `--ag` 등) 런타임 교체
